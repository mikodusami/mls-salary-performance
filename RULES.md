# The Rules

Commit every session without exception. Never delete locally without pushing first. When I feel the urge to expand scope — add a league, add a feature, add a new data source — write it in a `FUTURE.md` file instead. Get it out of my head and onto paper without breaking the project. If I get stuck on something for more than 30 minutes, move on and come back. Don't let one blocker kill a session.
